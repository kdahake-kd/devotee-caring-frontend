# ISKCON Logo

Please place the ISKCON logo image file here with the filename: `iskcon-logo.png`

## Instructions:
1. Download the ISKCON logo from the Pinterest link or official ISKCON website
2. Save it as `iskcon-logo.png` in this folder (`frontend/public/images/`)
3. The logo should be in PNG format with a transparent background (recommended)
4. Recommended size: 512x512 pixels or larger (the website will scale it appropriately)

## Alternative:
If you have the logo in a different format or name, you can:
- Rename it to `iskcon-logo.png`, or
- Update the image paths in the following files:
  - `frontend/src/components/Home.jsx`
  - `frontend/src/components/Dashboard.jsx`
  - `frontend/src/components/Analytics.jsx`
  - `frontend/src/components/SpiritualAnimations.jsx`
